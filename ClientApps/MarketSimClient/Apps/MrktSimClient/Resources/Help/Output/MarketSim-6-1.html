CHAPTER 3 Editing Marketing Plans and Scenarios

The MarketSim model is the composite of your data associated with products, segments, initial conditions, channels, product attributes, and social networks-all of which are critical to the formation of market plans.

Your MarketSim marketing plan is specific to a product, and defines the product's price, distribution, display, and media. From the marketing plan, you can create scenarios into which other factors can be applied, and on which various consumer behaviors and expectations can be simulated.

The Model and Marketing Plans controls are combined on a common window from which you can use tabs to switch the view from model parameters to marketing plan parameters. Use the following topics in the chapter to work with MarketSim models and marketing plans:

  • Working with Models
  • Working with Scenarios
  • Working with Market Plans

Working with Models

.Access modeling parameters by presenting the Model tab configuration panel from the MarketSim Model Editor.

Navigation: From MarketSim front page-->Database <name>--> Project: Models <click to select one> --> Model, Edit Model --> MarketSim Model Editor, Model tab = Model configuration panel

Products

Settings for brands and products define all products that will be included in your marketing scenario, as associated with the brand that sells each product. MarketSim imposes no limits on the number of products that you can define. Additionally, a brand can sell multiple products.

Settings for segments define how consumers select a channel from which to buy.

  • Network Models: Metrics that model consumers influenced by a product category, who are part of a large social network that is very persuasive.

Settings for initial conditions define the historic background of the market, prior to start of a scenario simulation.

Settings for attributes define characteristics that appeal to consumers, and assign ratings for the characteristics. MarketSim imposes no limitations on how you define attributes: they may be qualitative (Brand Perception) or quantitative (weight in grams). Some attributes may apply to all products in the scenario, while other attributes may apply to some products and be irrelevant to others.

Working with Scenarios

When editing a MarketSim model, you use the Scenario menu and drop-down selector to manage the names and ingredients for scenarios associated with the model.

Note: After a scenario is established by using the Model Editor, it must be further configured for simulation, as described in "Scenario Simulation Settings" on page 39.

Creating and Editing Scenarios

You set basic Scenario parameters by using the Create or Edit functions from the MarketSim Model Editor. This allows you to set the name, description, and components to be used for a particular scenario.

Navigation: From MarketSim front page-->Database <name>--> Project: Models <click to select one> --> Model, Edit Model --> MarketSim Model Editor, Model tab = Model configuration panel --> Scenario, New or Edit <name of scenario listed in the drop-down selector>

The fields you use to manage Scenario settings are described in "Creating and Editing Scenarios" on page 22.

Copying a Scenario

Copying a scenario replicates a specified scenario on the database. The name of the copied scenario is appended by a value of 1 after you confirm that you do want to proceed with the copy action.

You may want to use the copy function to apply an existing scenario as a `template' upon which you will apply modifications.

Navigation: From MarketSim front page-->Database <name>--> Project: Models <click to select one> --> Model, Edit Model --> MarketSim Model Editor, Model tab = Model configuration panel --> Scenario, Copy <name of scenario listed in the drop-down selector>

Deleting a Scenario

Deleting a scenario removes it from the database currently in use. MarketSim queries for confirmation of the deletion prior to removing the specified scenario.

Navigation: From MarketSim front page-->Database <name>--> Project: Models <click to select one> --> Model, Edit Model --> MarketSim Model Editor, Model tab = Model configuration panel --> Scenario, Delete <name of scenario listed in the drop-down selector>

Creating or Editing a Marketing Plan

A task sequence for Market Plan management, using the structure of the MarketSim Model Editor as a guide, is similar to the following steps that are described in this section:

1. Identify the scenario you want to include in the Marketing Plan.

2. Identify the Market Plan you want to include in the Marketing Plan.

3. Identify the Market Plans Components you want to include in the Marketing Plan.

4. Set Dates for the Market Plan.

5. Set Channel(s) for the Market Plan.

6. Identify the Product Plans to be in included in the Market Plan

7. Specify how you want data to be presented in this window.

Procedure

1. Identify the scenario you want to include in the Marketing Plan.

Use the Scenario menu to set up, copy, or delete scenarios. To delete or copy a scenario, you must first identify the scenario. To do so, choose a scenario from the drop-down selector, then issue the delete or copy command from the Scenario menu.

You can also use all scenarios in your marketing plan.

2. Identify the Market Plan you want to include in the Marketing Plan.

Use the Market Plans menu to import a .xls file, set up a new Market Plan. If a market plan is in view on the MarketSim Model Editor, you can use the Copy, Edit, or Delete options of the MarketPlans menu.

You can elect to use all market plans and/or External factors associated with this project. To do so, click to checkmark the Select All box and/or the External Factors box.

3. Identify the Market Plans Components you want to include in the Marketing Plan.

Use the Components drop-down selector to identify the component type, or select all to use all component types. Use the Components menu to import components from .xls files, or to copy, edit, set parameters, or delete a selected component.

Table 16 Market Plan Components
Component
Description
Price

Settings for price define the price plan that you will use with one or more marketing plans.

You can also modify price plan settings, copy the price plan for use in other scenarios, or delete the price plan.

Data Source: Data template.
Media

Settings for media define the media plan that you will use with one or more marketing plans.

You can also modify media plan settings, copy the media plan for use in other scenarios, or delete the media plan.

Data Source: Data template,
Coupons

Settings for coupons define the coupon plan that you will use with one or more marketing plans. Coupon data is part of the mass media data sheet.

You can also modify coupon plan settings, copy the coupon plan for use in other scenarios, or delete the coupon plan.

Data Source: Data template.
Display

Settings for display define the display plan you will use with one or more marketing plans. A display plan defines how a consumer may be likely to experience a special promotional display in a store during a shopping trip.

Data Source: Data template
Distribution

Settings for distribution define the distribution plan you will use with one or more marketing plans. These settings model values by which distribution persuades users. Persuasion can be either positive or negative.

Data Source: Data template
External Factors

External Factors are a bit different from other component plans because they are associated directly with a scenario rather than a market plan, and appear in the Market Plans list rather than the Components list (if the External Factors checkbox is checked). External Factors are imported in the same way as are Components (using the popup menu for Components).

You can modify external factors plan settings, copy the external factors plan for use in other scenarios, or delete the external factors plan.

Data Source: Data template.

4. Set Dates for the Market Plan.

Use the Dates drop-down menu to choose either all Dates, a Range of Dates, a specified Date, a previous Date, or a forthcoming Date.

5. Set Channel(s) for the Market Plan.

Use the Channel drop-down menu to choose either all Channels or a specified channel. The channels available are the result of configuration for Channels at the Model window of the MarketSim Model Editor.

6. Identify the Product Plans to be in included in the Market Plan

At the Brand/Project Product Hierarchy, click to checkmark the products to be included in the Market Plan.

7. Specify how you want data to be presented in this window.

At the Data menu, indicate your preference for auto update as either turned on or turned off. When on, the Refresh Now button will flag you to press it whenever MarketSim needs synchronization of new settings for the Market Plan.

At the Graph check box, click to checkmark it if you want to see your data graphically represented. When unchecked, the data is presented in tabular format.
