CHAPTER 5 Viewing Simulation Results

Use guidelines in this chapter to use results of your simulations, as described in the following topics:

  • Viewing Results
  • Deleting Results
  • Creating Charts
  • Viewing Warnings

Note: Results of a simulation cannot be viewed until a scenario has successfully completed a simulation run.

Viewing Results

Once you have generated results from a simulation, you can save the simulation file to a.csv file and/or create charts to graphically depict the peaks and valleys of the various components in your scenario.

Simulation results tools are contained in a 3-part window, which provide the following tab panels:

  • Time Series Charts
  • Summary Tables
  • Variables

Summary Tables

Use the Summary Tables window to Generate summary table views associated with Simulation and/or Calibration. The Summary Table enables you to specify metrics to be computed for the entire running simulation. Generated summaries can be saved as .csv files for use in creating additional data sheets.

Summary tables provide comparison of the simulated and total sales as based on number of units and market share, using the following conventions to compute the difference:

  • The percentage of error for the total time simulated
  • A root-mean-square (RMS) of the sum of the difference for each week between real units sold and simulated units
  • The Mean Absolute Percentage Error (MAPE) calculation. This calculation is the absolute value of the difference between the simulated sales and the actual sales for the same weekly period, divided by the actual sales for that period. The mean-percentage-difference is calculated over the whole test period to obtain a single value that indicates goodness of fit.
  • Total dollar volume of units sold during the simulation

This comparison can be divided into two parts-a calibration time period and a testing time period. This allows you to tune the parameters of a scenario to optimize the fit (minimize the error) over the calibration time period, and see if the fit also improves during the testing time period.

To obtain good results, make sure to set dates as follows:

  • Start date of each table is on or after the start date of the simulation.
  • End date of each table is on or before the end date of the simulation.

Note: You do not need to use the second table-you can gave the first table cover the entire time period of the simulation.

Consider using one table for the calibration time period, and the other table for the testing time period. The calculations of the spreadsheet do not depend on how you interpret these tables.

Deleting Results

Use the Delete Results option to remove any unneeded results, to save space on your computer.

Navigation: Scenario/Simulations/Results window: Results menu, Delete Results

There is no confirmation query when deleting results.

Creating Charts

1. Present the 3-tab Results window.

2. Click Refresh List to update the simulation list.

3. Set the Time span to indicate how you want the data to be aggregated into the chart.

4. Set Scope, using the Time Series Charts selector and/or the Summary Tables selector. Use the Segment and Channel drop down selectors to define each to use all or specific components.

5. Set Graph options.

6. Set Available Data: use the Data type drop-down selector to set available data from marketing input, simulation results, or real sales, and move the components you want to include into the right-most pane.

7. Click the Save Setting button, then the Create Charts button to see your data plotted to a chart.

Viewing Warnings

Problems encountered when running simulations are reported by MarketSim in warning messages. To view the messages, use the Results menu in the Scenario/Simulations/Results window to present the Sim Log.
